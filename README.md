# RealNumberApp (WPF, MVVM)

## Описание

Приложение реализует класс «Вещественное число»:

* знак
* мантисса
* порядок

Форма записи:
±M * 10^E

## Возможности

* Сложение
* Вычитание
* Умножение
* Деление
* Нормализация

## Архитектура

MVVM:

* Models
* ViewModels
* Views

## Запуск

dotnet build
dotnet run
